## 项目功能
获取odt、ods、docx、xlsx中的超链接。通过**cloudmersive** 

## 部署
`pip3 install -r requirements.txt`

`main.py` 为命令行版。

`main_gui.py` 为通过 [Geooey](https://github.com/chriskiehl/Gooey) 生成的桌面版本，通过 `pyinstaller` 打包生成桌面包。

## 使用 

`python3 main.py -d /to/path/`
